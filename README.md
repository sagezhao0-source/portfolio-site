# Smart Learner（本地移动端 MVP）

本项目是 **纯移动端本地版**：无后端、无账号、数据保存在本机；到点通过 **本地通知**提醒，并在提醒页给出 **3 条 Hooks + 1 条 Next step**，完成「立即开始 / 稍后再说 / 不再学习」闭环。

## 你需要先安装

- Node.js（建议装最新版 LTS，安装后应能在终端运行 `node -v` / `npm -v`）
- （可选）Android Studio / Xcode：真机或模拟器运行

## 安装与运行

在项目根目录：

```bash
npm install
npx expo start
```

## 功能清单（MVP）

- 粘贴链接收藏，选择提醒时间（预设：2 小时后 / 今天 21:00 / 明天同时间 / 自定义）
- 后台生成：
  - 3 条学习动力钩子（Hooks，非枯燥摘要）
  - 1 条最小可执行动作（Next step）
- 到点本地通知提醒
- 提醒页三选一：
  - 立即开始：打开原链接，并记录开始事件
  - 稍后再说：自动顺延（默认 +2 小时；可改为明天同时间）
  - 不再学习：移入已放弃归档（不再提醒）

## 说明

本版本的 Hooks 生成采用 **轻量启发式**（基于标题/域名/URL 关键词），不依赖大模型；后续若要接入真实 LLM，只需替换 `src/ai/generateHooks.ts` 的实现即可，UI 与流程不需要重做。
